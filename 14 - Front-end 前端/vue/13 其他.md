## 其他

+ watch 

    监听data中数据的变化
    
    监听路由地址的改变

```html
<div id="app">
    <input type="text" v-model="name">
</div>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            name:'john'
        },
        watch:{
            'name':function(newVal,oldVal) {
                console.log(oldVal + "->" + newVal)
            },
            '$route.path':function(newVal,oldVal) {
                console.log(oldVal + "->" + newVal)
            }
        }
    })
</script>
```

+ computed

    使用时作为普通属性引用
    
    当计算属性依赖的响应式属性改变时，会进行重新计算
    
    计算属性的结果会被缓存，可直接使用
```html
<div id="app">
    <input type="text" v-model="name">
</div>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            name:'john'
        },
        watch:{
            'name':function(newVal,oldVal) {
                console.log(oldVal + "->" + newVal)
            },
            '$route.path':function(newVal,oldVal) {
                console.log(oldVal + "->" + newVal)
            }
        }
    })
</script>
```
